# 英文单词与句子复习工具 — 设计文档

日期:2026-06-23
面向用户:一年级和四年级的两个孩子

## 1. 目标

做一个跨平台的互动式 HTML 工具,帮孩子复习英文单词、短语、句子,
同时熟悉键盘布局。词库由 Claude 离线从课本照片/文件解析生成。

核心价值:复习内容 + 练键盘 + 错题强化,三合一,零安装、双击即用。

## 2. 整体架构

由两部分组成:

1. **词库文件(JSON)** —— Claude 离线生成。家长把课本照片/文件发给 Claude,
   Claude 识别单词、短语、句子,补全中文释义、音标、例句,产出一个 `.json` 词库文件。
   每个文件对应一个孩子的一个单元/课文。

2. **复习工具(单个 HTML 文件)** —— 自包含的 `index.html`(内嵌 CSS/JS,
   不依赖网络与框架)。双击即可在任意电脑/平板浏览器打开,跨平台。
   词库文件**单独存放**,在工具里点击"加载词库"导入。

### 数据流

```
课本照片 ──(Claude 解析)──> 词库.json ──(工具内点击加载)──> HTML 工具
                                                              │
                                      闯关进度 + 错题本 + 统计 ──> 浏览器 localStorage
                                                              │
                                                      导出/导入备份文件(JSON)
```

### 存储策略

- 进度、错题本、作答时间统计 全部存在浏览器 `localStorage`。
- 提供"导出备份 / 导入备份"按钮,生成/读取一个 JSON 文件,方便换设备迁移。
- 数据组织为两级:`孩子 → 词库`。每个词库独立记录进度与错题本。

## 3. 词库文件格式

```json
{
  "meta": {
    "child": "哥哥",
    "title": "四年级上 Unit 5",
    "grade": 4
  },
  "items": [
    {
      "word": "library",
      "meaning": "图书馆",
      "ipa": "/ˈlaɪbreri/",
      "type": "word",
      "example": "I borrow books from the library.",
      "exampleMeaning": "我从图书馆借书。"
    },
    {
      "word": "get up",
      "meaning": "起床",
      "ipa": "/ɡet ʌp/",
      "type": "phrase",
      "example": "I get up at seven.",
      "exampleMeaning": "我七点起床。"
    },
    {
      "word": "How are you?",
      "meaning": "你好吗?",
      "ipa": "",
      "type": "sentence",
      "example": "",
      "exampleMeaning": ""
    }
  ]
}
```

字段说明:
- `type`:`word`(单词)/ `phrase`(短语)/ `sentence`(句子)。决定该条目可进入哪些模式。
- `grade`:年级,驱动难度自适应。
- `ipa`:音标,播放发音时同屏显示。句子可留空。
- `example` / `exampleMeaning`:例句及其中文,是完形填空的核心素材。

## 4. 发音

- 方案:**音标 + 浏览器朝读**。
- 用浏览器自带的 Web Speech API(`speechSynthesis`)朝读单词/短语/句子,零成本、离线可用、跨平台。
- 播放时同屏显示 `ipa` 音标。
- 每道需要理解词义的题旁都有"🔊 发音"按钮。

## 5. 闯关模式(5 种)+ 难度自适应

通用规则:
- 每关固定题数,开始前可选 **5 / 10 / 15** 题(默认 10)。
- 从所选词库(或错题本)随机抽题。
- 通关后显示成绩:答对题数、用时、星级(全对 3 星,递减)。
- 难度按词库 `grade` 自动调整。

### ① 拼写闯关 1 — 补全单词
显示单词但隐去部分字母(如 `l _ b r _ _ y`),下方给中文释义 + 音标 + 发音。
孩子用键盘敲字母补全空缺。
- 一年级:隐去约 1/3,偏后半段;仅 `word` 类。
- 四年级:隐去过半,随机位置;含 `word` + `phrase`。

### ② 拼写闯关 2 — 全词默写
只显示与字母数对应的空格(`_ _ _ _ _ _ _`)+ 中文释义 + 发音。
孩子敲完整单词。
- 一年级:优先短单词(≤5 字母),可点"提示"露出首字母。
- 四年级:全部单词 / 短语。

### ③ 意思闯关 1 — 看词选意思
给英文单词,4 个中文选项选 1。键盘按 `1/2/3/4` 选择(练数字键)。

### ④ 意思闯关 2 — 看意思选词
给中文释义,4 个英文单词选 1。键盘 `1/2/3/4` 选择。

### ⑤ 完形填空闯关 — 句中敲词
取例句/句子,挖掉核心单词:`I borrow books from the _______.`,
下方给中文整句提示。孩子敲出缺失单词。
- 一年级:仅用短例句,挖 1 个简单词。
- 四年级:用完整句子,可挖较长的词。

### 键盘练习(贯穿全程)
- 需要敲字母的模式(①②⑤):屏幕底部显示**虚拟键盘**,
  下一个该敲的键**高亮**;敲对亮绿、敲错亮红并抖动。
- 选择题模式(③④):提示数字键 `1-4`。
- 目的:让孩子看着键盘逐渐记住键位,从看键过渡到盲打。

## 6. 错题本

- **自动收集**,两种入库条件:
  1. 第一次答错的题。
  2. 答对但**耗时异常长**的题(见下方统计规则)——抓"勉强会但不熟"的词。
- **专项练习**:主菜单"错题本"入口,只从错题抽题,同样支持 5 种模式。
- **移出规则**:某错题在专项练习中**连续答对 3 次**,自动移出错题本。
- 错题本按"孩子 + 词库"分别存储,随 localStorage 与导出/导入备份。

### "耗时异常长"的判定

- 对**每种题型**分别累计孩子的历史作答时间(只统计答对的题),
  计算平均值 μ 和标准差 σ。
- 某题答对,但耗时 > **μ + 1σ**,即使答对也加入错题本。
- **冷启动**:样本不足 10 条时 μ/σ 不可靠,暂不触发"超时入库",
  仅按"答错入库";攒够 10 个样本后启用统计判断。
- 统计数据按"孩子 + 题型"长期累计,存 localStorage,随备份迁移,越用越准。

## 7. 界面风格

- **活泼游戏风**:鲜艳配色、大按钮、星星/金币/过关动画、音效反馈。
- 主流程:加载词库 → 选孩子 → 选词库 → 选模式 → 选题数 → 闯关 → 成绩页。
- 适配电脑与平板触屏(虚拟键盘也可点击,但鼓励用物理键盘)。

## 8. 文件结构

```
word-review-tool/
├── index.html              # 复习工具(自包含)
├── wordbanks/              # 词库文件目录(Claude 生成的 JSON 放这里)
│   ├── example-grade1.json
│   └── example-grade4.json
└── docs/superpowers/specs/ # 设计文档
```

## 9. 非目标(YAGNI)

- 不做云端账号/同步(用导出/导入代替)。
- 不在 HTML 内做照片 OCR(由 Claude 离线解析)。
- 不生成音频文件(用浏览器朝读 + 音标代替)。
- 不做联机排行榜、社交等附加功能。
