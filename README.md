# 🎮 单词闯关乐园

给孩子用的英文单词 / 句子复习工具,边复习边练键盘。打开即用,跨平台(电脑、平板都行)。

## 两个版本

本仓库有两个 HTML,按用途选:

- **`单词闯关开发版.html`** —— 可加载任意 JSON / PDF 词库,用于开发、调试、生成新词库。本地开发用。
- **`单词闯关四年级通关版.html`** —— 内嵌四年级全部词库(236 条)的固化单文件,双击即用,**用于分享给别人**(孩子 / 家长)。无需加载,打开就有内容。

## 怎么用

1. **双击对应的 HTML** —— 用任意浏览器(Chrome / Edge / Safari)打开,不用装任何东西。
   - 分享给别人:发 `单词闯关四年级通关版.html`,对方双击即用。
   - 本地开发:用 `单词闯关开发版.html`。
2. **加载词库**(仅开发版)—— 点首页右上角「＋ 加载词库」,选择 JSON 或 PDF 文件。
   - 已附两个示例词库:`example-grade1.json`(一年级)、`example-grade4.json`(四年级)。
   - **每加载一个文件 = 一个「子库」**。加载过的词库会被记住,下次自动出现。
3. **选词库 → 选模式 → 闯关**。

## 子库 / 总库

- **子库**:每个文件就是一个子库(如「四年级下 Unit 5」),可单独选它专项练习。
- **总库**:同一个孩子名下有 2 个及以上子库时,会自动多出一张「📚 总库」卡片 ——
  把他所有子库**合并**一起练。不同孩子(不同年级)不会混在一起。

## 加载 PDF

「＋ 加载词库」也接受 PDF:

- **文字版 PDF(可选中文字)**:浏览器直接抽取文字,弹出确认框,可编辑后生成子库。
  音标、例句不会自动有 —— 这些发给 Claude 生成精品词库时才补全。
- **扫描 / 拍照版 PDF**:里面是图片,浏览器识别不了,会提示你把 PDF 发给 Claude。
- pdf.js 已**本地打包**在 `lib/` 目录,完全离线可用。

> 想要带准确释义、音标、例句的**精品词库**:把课本照片或 PDF 发给 Claude,
> 生成标准 JSON 后加载即可,质量最高。

## 五种闯关模式

| 模式 | 玩法 |
|------|------|
| ✏️ 拼写闯关 1 | 单词隐去部分字母,看中文敲字母补全 |
| ⌨️ 拼写闯关 2 | 只给空格和中文,默写整个单词 |
| 🤔 意思闯关 1 | 看英文单词,按 1/2/3/4 选中文意思 |
| 🔤 意思闯关 2 | 看中文意思,按 1/2/3/4 选英文单词 |
| 📝 完形填空 | 句子里挖掉一个词,根据句意敲出来 |

- **难度自适应**:词库里写了 `grade`,一年级隐去字母少、给更多提示;四年级更难。
- **虚拟键盘**:屏幕下方画出键盘,下一个该敲的键会高亮,帮孩子记键位。
- **发音**:用浏览器朗读单词/句子,旁边显示音标。

## 错题本(分子库 / 总库)

- 第一次没答对、**或答对但太慢**的题,会自动进错题本。
- 错题始终记在它原本的子库里。
  - 在**子库**里点「📒 子库错题本」:只练这个子库的错题。
  - 在**总库**里点「📒 总库错题本」:把这个孩子所有子库的错题合在一起练。
- 某道错题**连续答对 3 次**就自动毕业、移出错题本。

## 进度与备份

- 闯关进度、错题本、统计都存在本机浏览器里。
- 换设备时:旧设备点「📤 导出备份」存成一个文件,新设备点「📥 导入备份」读进去。

## 怎么加新词库

把课本照片或单词表发给 Claude,Claude 会生成一个新的 `.json` 词库文件,
放进 `wordbanks/` 文件夹,在工具里加载即可。词库格式见 `docs/superpowers/specs/` 里的设计文档。
