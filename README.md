# 🎮 单词闯关乐园

给孩子用的英文单词 / 句子复习工具,边复习边练键盘。打开即用,跨平台(电脑、平板都行)。

## 怎么用

1. **双击 `index.html`** —— 用任意浏览器(Chrome / Edge / Safari)打开,不用装任何东西。
2. **加载词库** —— 点首页右上角「➕ 加载词库」,选择 `wordbanks/` 里的 JSON 文件。
   - 已附两个示例词库:`example-grade1.json`(一年级)、`example-grade4.json`(四年级)。
   - 加载过的词库会被记住,下次打开自动出现。
3. **选词库 → 选模式 → 闯关**。

## 五种闯关模式

| 模式 | 玩法 |
|------|------|
| ✏️ 拼写闯关 1 | 单词隐去部分字母,看中文敲字母补全 |
| ⌨️ 拼写闯关 2 | 只给空格和中文,默写整个单词 |
| 🤔 意思闯关 1 | 看英文单词,按 1/2/3/4 选中文意思 |
| 🔤 意思闯关 2 | 看中文意思,按 1/2/3/4 选英文单词 |
| 📝 完形填空 | 句子里挖掉一个词,根据句意敲出来 |

- **难度自适应**:词库里写了 `grade`,一年级隐去字母少、给更多提示;四年级更难。
- **虚拟键盘**:屏幕下方画出键盘,下一个该敲的键会高亮,帮孩子记键位。
- **发音**:用浏览器朗读单词/句子,旁边显示音标。

## 错题本

- 第一次没答对、**或答对但太慢**的题,会自动进错题本。
- 主菜单点「📒 练习错题本」专项练习。
- 某道错题**连续答对 3 次**就自动毕业、移出错题本。

## 进度与备份

- 闯关进度、错题本、统计都存在本机浏览器里。
- 换设备时:旧设备点「📤 导出备份」存成一个文件,新设备点「📥 导入备份」读进去。

## 怎么加新词库

把课本照片或单词表发给 Claude,Claude 会生成一个新的 `.json` 词库文件,
放进 `wordbanks/` 文件夹,在工具里加载即可。词库格式见 `docs/superpowers/specs/` 里的设计文档。
